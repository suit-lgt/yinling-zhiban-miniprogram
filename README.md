## 项目简介
面向老年人的一站式适老化服务小程序，解决数字鸿沟问题，核心功能包括：
- 语音优先的适老化交互（支持方言识别、3步内完成核心操作）；
- 健康数据监测与异常提醒（血压/血糖记录+7天趋势分析）；
- 紧急求助SOS（一键呼叫+位置共享）；
- AI智能咨询（反诈提醒、养老政策解答）。

## 技术亮点
1. 多引擎语音融合：科大讯飞+百度语音双引擎加权，提升老年人口音识别准确率；
2. 上下文意图解析：理解老年人隐含需求（如“头晕”→自动跳转血压记录）；
3. 轻量化架构：微信小程序+本地存储，适配老年机性能，零服务器成本；
4. 适老化设计：大字号、高对比度、容错提示，符合HIG适老化标准。
